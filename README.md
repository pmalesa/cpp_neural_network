# CPP Neural Network
A library for creating simple neural networks written in C++.

## Install library system-wide
```bash
sudo ninja install
```

## Uninstall library
```bash
sudo ninja uninstall
```

## Tests
### Compile binary for running unit tests
```bash
ninja

# or explicitly
ninja run_tests
```

### Run unit tests
```bash
./run_tests
```

### Clean
```bash
ninja clean
```
